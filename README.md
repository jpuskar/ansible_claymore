Role Name
=========

Configures AMDGPU-PRO, ROCm, and Claymore for Ubuntu.


Requirements
------------

- Python installed.
- Pre-stage amdgpu-pro-17.40-483984.tar.xz at /root.
- Secure-boot must be disabled.


Role Variables
--------------

- wallet_address
- wallet_email
- miner_name
